# 环境变量快速参考

- `VTF_OUTPUT_SINK`: 默认 sink (markdown / feishu)
- `VTF_TRANSCRIBE_FUNASR_PYTHON`: FunASR Python 路径
- `VTF_TRANSCRIBE_ASR_MODEL`: ASR 模型 (默认 paraformer-zh)
- `VTF_PLATFORM_BILIBILI_COOKIES_FROM_BROWSER`: B站 cookie 浏览器 (chrome)
- `VTF_SINK_FEISHU_BASE_TOKEN`: 飞书表格 token
- `VTF_SINK_FEISHU_TABLE_ID`: 飞书表格 ID
- `VTF_SINK_FEISHU_SCHEMA`: 字段 schema 文件路径
